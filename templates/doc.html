/api/installations get

get

Liste de toutes les installations ordonnées par nom, qui correspondent aux critères de requête s'ils sont fournis, sinon toute les installations existante.

get /api/installations

Liste de toutes les installations ordonnées par nom, qui correspondent aux critères de requête s'ils sont fournis, sinon toute les installations existante.

  • Request
  • Response

Query Parameters

  • arrondissement: (string)
  • nom: (string)

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: Installation

  • nom: required(string)
  • type: required(string)
  • arrondissement: required(string)
  • ouvert: required(boolean)
  • deblaye: required(boolean)
  • condition: required(string)
  • mise_a_jour: required(string)
  • arrose: required(boolean)
  • resurface: required(boolean)
  • id_uev: required(number)
  • type_piscine: required(string)
  • adresse: required(string)
  • propriete: required(string)
  • gestion: required(string)
  • equipement: required(string)
  • point_x: required(string)
  • point_y: required(string)
  • longitude: required(number)
  • latitude: required(number)
